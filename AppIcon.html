How to Generate Icons:

  1. Choose your preferred text placement
  2. Click "Download Icon" to save the 1024x1024 PNG
  3. Run ./generate_icons.sh in terminal to create all sizes
  4. Drag the generated icons from AppIcons folder into Xcode
